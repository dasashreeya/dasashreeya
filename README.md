### Agentic RAG System  
Self-correcting retrieval-augmented generation system using LangGraph, LangChain, and OpenAI.

- Autonomous query planning + retrieval  
- Relevance grading and iterative refinement  
- Human feedback logging for improvement  
- Streamlit interface with backend storage

### Statistical Domain Language Model  
GPT-style transformer trained from scratch on statistical textbook data.

- Built full preprocessing and training pipeline  
- Domain-specific corpus creation from PDFs  
- Tokenization, deduplication, and training  
- Implemented using PyTorch + Hugging Face  

🔗 https://github.com/dasashreeya/statistical-domain-language-model
